LUMEN is a fully local, modular AI desktop assistant built with Python.
Voice-controlled with an intent-action system.

## Features
- Open applications
- Send messages (WhatsApp)
- Web search (DuckDuckGo, no API key needed)
- Weather reports
- Temporary & long-term memory
- Conversational AI with context

## Stack
- Speech-to-Text: Vosk (offline)
- Text-to-Speech: Piper TTS (offline)
- LLM: LM Studio (local, OpenAI-compatible API)
- UI: Tkinter + Pillow
- Search: DuckDuckGo

## Security
- Runs 100% locally
- No remote access by default
- LUMEN only executes what YOU code
